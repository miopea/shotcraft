## Why Shotcraft

Existing tools (screenshots.pro, Bannerbear, Placid) all require uploaded
screenshots. **Shotcraft captures from your running app.** Point it at a
dev server or staging URL, give it your auth flow, and one command later
you have a complete marketing screenshot set.

## Quickstart

```bash
pnpm add -D shotcraft @shotcraft/template-app-store-iphone @shotcraft/template-readme-hero
pnpm shotcraft init       # scaffold shotcraft.config.ts
pnpm shotcraft doctor     # sanity-check the config
pnpm shotcraft            # capture + render end-to-end
```

PNGs land in `./screenshots/{template-id}/{name}-{theme}.png`.
